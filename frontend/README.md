# Frontend - Endorse Translate

Frontend en React para consumir el API de traducción de endosos.

## Instalación

```bash
npm install
```

## Configuración

Crea un archivo `.env` en la raíz del proyecto frontend:

```
REACT_APP_API_URL=http://localhost:3000/v1
```

## Ejecutar

```bash
npm start
```

La aplicación se abrirá en `http://localhost:3000` (o el puerto disponible).

## Características

- Autenticación JWT
- Formulario para ingresar datos de endoso
- Visualización del JSON estructurado resultante
- Copiar JSON al portapapeles
- Diseño responsive
